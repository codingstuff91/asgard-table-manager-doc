# Restriction de session

> Les utilisateurs possédant le rôle administrateur ont la possibilité de restreindre une session.

Pour restreindre une session, il faudra cliquer sur l'icone jaune située sur la session concernée.

<img class="block m-auto" src="/assets/img/restriction-session.png" alt="Screenshot" />

Ensuite il faudra fournir une justification concernant la restriction de capacité en salles de la session.

Une fois le formulaire validé, un message informatif apparaitra aux utilisateurs les informant de la réduction en capacité de salles.
